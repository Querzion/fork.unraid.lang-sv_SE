Hide Deprecated Applications=Göm föråldrade applikationer
This setting hides deprecated applications from the list of available apps. Deprecated apps will still, however, display within the installed and previous apps sections=Denna inställning gömmer föråldrade applikationer från listan över tillgängliga appar. Föråldrade appar kommer dock fortfarande att visas inom sektionerna installerade och tidigare appar

Hide Incompatible Applications=Göm inkompatibla applikationer
This setting will hide applications that are listed as being incompatible with your version of Unraid=Denna inställning döljer applikationer som är listade som inkompatibla med din version av Unraid

Allow install of second instance=Tillåt installation av andra instans
This option enables another button on the cards and popups which will allow you to reinstall an installed application using its default template values. This option can be confusing at times and is only generally used if you wish to install and have an application running twice simultaneously=Detta alternativ aktiverar en annan knapp på kort och popups som gör att du kan installera om en installerad applikation med standardmallvärden. Detta alternativ kan vara förvirrande ibland och används vanligtvis bara om du vill installera och ha en applikation som körs två gånger samtidigt

Automatically open the sidebar=Öppna sidofältet automatiskt
Allows CA to automatically open the sidebar for more information when hovering over an application's icon=Tillåter CA att automatiskt öppna sidofältet för mer information när du hovrar över en applikations ikon

Allow CA to check for updates to applications=Tillåt CA att kontrollera uppdateringar för applikationer
This setting enables CA to check for updates to applications. Note that Community Applications will always check for updates to itself=Denna inställning tillåter CA att kontrollera uppdateringar för applikationer. Observera att 'Community Applications' alltid kommer att kontrollera uppdateringar för sig själv

Disable Mouse Wheel Page Changes=Inaktivera musrullningssidesbyten
This setting disables using left or right on a mouse wheel to change the page=Denna inställning inaktiverar användningen av vänster eller höger på musrullningen för att byta sida

Allow CA to send any emergency notifications=Tillåt CA att skicka eventuella nödsignaler
This setting allows CA to periodically scan your system and alert you to the presence of
1 Known malware and security vulnerabilities within applications=1 Känd skadlig programvara och säkerhetsbrister inom applikationer
2 Critical security vulnerabilities within Unraid OS=2 Kritiska säkerhetsbrister inom Unraid OS
3 Any installed applications which may seriously hinder the operation of your server=3 Eventuella installerade applikationer som kan allvarligt hindra driften av din server

Save CA debugging information=Spara CA felsökningsinformation
Only enable this when advised to from the author of Community Applications=Aktivera endast detta när du fått råd om det från författaren till Community Applications

Enable developer mode=Aktivera utvecklarläge
This unlocks some extra features within CA generally only used by developers. Not recommended to enable=Detta låser upp några extra funktioner inom CA som vanligtvis bara används av utvecklare. Rekommenderas inte att aktivera
